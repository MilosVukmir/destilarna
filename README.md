# Destilarna

Informacijski sistem za upravljanje delovanjem destilarne.

## Opis projekta

Projekt predstavlja informacijski sistem za podporo procesom v destilarni. Sistem omogoča upravljanje dobaviteljev, prevzema sadja, fermentacije, destilacije, serij žganja, sodov, strank in prodaje.

## Glavne funkcionalnosti

* Upravljanje dobaviteljev
* Evidentiranje prevzema sadja
* Spremljanje fermentacije
* Evidentiranje destilacije
* Upravljanje serij žganja
* Upravljanje sodov
* Evidentiranje prodaje
* Upravljanje strank
